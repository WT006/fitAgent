# fitagent 前端快速启动说明

## 环境要求

| 工具 | 版本要求 |
|------|----------|
| Node.js | >= 18 |
| npm | >= 9 |
| Git Bash | Windows 推荐（也可使用 PowerShell） |
| 后端服务 | Spring Boot 运行在 `http://localhost:8122` |

## 1. 安装依赖

在项目根目录 `fit-ai-agent-frontend` 下执行：

```bash
npm install
```

## 2. 启动后端

确保 Spring Boot 后端已启动，API 地址为：

```
http://localhost:8122/api
```

可访问 Swagger 文档验证：`http://localhost:8122/api/doc.html`

## 3. 启动前端开发服务器

```bash
npm run dev
```

启动后访问：**http://localhost:5173**

前端通过 Vite 代理将 `/api` 请求转发至 `http://localhost:8122`，无需额外配置 CORS。

## 4. 功能验证流程

### 新用户注册登录（AC-01）

1. 访问 http://localhost:5173/register
2. 填写账号（4-16 位字母/数字/下划线）、密码（至少 8 位）
3. 注册成功后跳转登录页，登录后自动进入打卡页

### AI 规划（AC-02）

1. 在打卡页点击「AI 帮我规划」
2. 依次回答 4 个固定问题
3. 等待 AI 生成 30 天计划预览（约 10-30 秒）
4. 可编辑任务标题，点击「确认计划」
5. 自动跳转回打卡页

### 每日打卡（AC-03 / AC-04 / AC-05）

- **今天**：查看任务并打勾
- **过去日期**：切换日期后可补打勾
- **未来日期**：可预览任务，checkbox 禁用不可打勾

### 重新规划（AC-06）

1. 打卡页点击「AI 帮我规划」
2. 确认弹窗后清除旧计划
3. 重新走问答流程

### AI 对话

- 导航栏「AI 对话」进入
- **健康咨询** Tab：FitApp RAG 运动健康问答（SSE 流式）
- **智能体** Tab：Manus 超级智能体（单轮任务）

## 5. 生产构建

```bash
npm run build
npm run preview
```

构建产物在 `dist/` 目录。

## 6. 常见问题

| 问题 | 解决方案 |
|------|----------|
| 接口 401 / 登录失效 | 重新登录；Token 有效期 7 天 |
| 规划生成超时 | 后端 AI 调用较慢，等待 30 秒或重试 |
| 接口连接失败 | 确认后端 `8122` 端口已启动 |
| 规划会话过期 | Redis 会话 2 小时过期，需重新开始规划 |

## 7. 项目结构概览

```
src/
├── api/          # Axios 封装 + 接口定义
├── components/   # UI 组件（打卡、对话、规划、布局）
├── composables/  # 组合式逻辑（SSE、规划状态机）
├── stores/       # Pinia 状态（auth、plan、chat）
├── views/        # 页面视图
└── router/       # 路由 + 导航守卫
```

## 8. 技术栈

- Vue 3 + TypeScript + Vite 6
- Pinia + Vue Router 4
- Tailwind CSS 4
- Axios + EventSource (SSE)
- Lucide Vue Next 图标
